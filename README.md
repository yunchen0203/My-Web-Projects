# My-Web-Projects
114 學年度資訊學科專題作品

## 📝 專案背景
本作品為高二資訊專題課程的期末成果。在掌握 HTML 與 CSS 的基礎語法後，我選擇挑戰將靜態網頁轉化為具備動態互動感的視覺藝術。本專案以「飲料與食物」為主題，透過精確的座標控制與動態時序設定，實作了具備層次感的網頁互動體驗。

## 🎨 創意核心：珍珠奶茶「動態加料」系統
這是本專案中我最引以為傲的技術突破。在完成老師指定的基礎飲品動畫之餘，我主動進行了功能延伸，將「沒料」的杯子升級為「加了珍珠」的珍珠奶茶。

* **多層次節點佈局**：在 HTML 中細緻地定義了 14 個珍珠節點（.bubble1 至 .bubble14），作為後續動態演算的基礎。
* **視覺邏輯補償**：利用 CSS :hover 偽類實作複雜的聯動效果。當使用者將滑鼠移入杯身時，系統會同步執行「飲料水位下降」與「珍珠群體升起」的視覺模擬，營造出飲用的真實感。

* **非同步位移**：我為每顆珍珠設定了獨立的 transition 延遲（從 0.8s 到 1.8s 不等）。
* **動態美學**：透過時間差的設計，讓珍珠升起時呈現出錯落有致的流體感，而非生硬的集體移動。

## 核心實作技術
* **幾何構造：`border` 邊框控制**
網頁原生元件多為矩形，我利用了 `border-top` 搭配左右邊框透明 (`border-left/right: solid transparent`) 的技巧，手繪出「上寬下窄」的梯形杯體 (`.cup`)。
* **空間布局：`position: absolute` 與 `margin: auto`**
透過絕對定位配合 `left: 0; right: 0`; `margin: auto`; 的技巧，精準讓杯子與吸管在容器中水平居中，並確保珍珠能鎖定在杯底的相對位置。
* **動態交互：:`hover` 與 `:focus`**
  * 使用 `:hover` 觸發整套「水位下降 + 珍珠噴發」的複合動畫。
  * 額外設定 `tabindex` 與 `:focus`，讓使用者點擊或使用鍵盤選取時也能觸發元件登場，實作「零 JavaScript」的動態切換。
* **視覺流暢度：`transition` 過渡屬性**
針對不同的屬性（如 `height`、`bottom`、`top`）設定特定的時間曲線，確保動畫過程平滑且具備物理感。
