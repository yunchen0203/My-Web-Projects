# My-Web-Projects
114 學年度資訊學科專題作品
## 📝 專案背景
本作品為高二資訊專題課程的期末成果。在掌握 HTML 與 CSS 的基礎語法後，我選擇挑戰將靜態網頁轉化為具備動態互動感的視覺藝術。本專案以「飲料與食物」為主題，透過精確的座標控制與動態時序設定，實作了具備層次感的網頁互動體驗。
## 🎨 創意核心：珍珠奶茶「動態加料」系統
這是本專案中我最引以為傲的技術突破。在完成老師指定的基礎飲品動畫之餘，我主動進行了功能延伸，將「沒料」的杯子升級為「加了珍珠」的珍珠奶茶。
### 技術實作：從「指定」到「原創」
* **多層次節點佈局**：在 HTML 中細緻地定義了 14 個珍珠節點（.bubble1 至 .bubble14），作為後續動態演算的基礎。
* **視覺邏輯補償**：利用 CSS :hover 偽類實作複雜的聯動效果。當使用者將滑鼠移入杯身時，系統會同步執行「飲料水位下降」與「珍珠群體升起」的視覺模擬，營造出飲用的真實感。
差異化動態演算法：

* **非同步位移**：我為每顆珍珠設定了獨立的 transition 延遲（從 0.8s 到 1.8s 不等）。

* **動態美學**：透過時間差的設計，讓珍珠升起時呈現出錯落有致的流體感，而非生硬的集體移動。
### 純 CSS 幾何繪圖
* **幾何構造**：不使用任何圖片，完全利用 border-top 與 transparent（透明邊框）技巧，在網頁上勾勒出梯形的幾何杯身（.cup & .inside）。

* **零 JS 交互**：利用 tabindex 屬性與 :focus 偽類，實作了無需 JavaScript 即可觸發的元件切換與登場效果。
## 🧠 程式邏輯
* **絕對定位 (Absolute Positioning)**：精確控制每一顆珍珠在容器內的初始與終止座標。

* **偽類選擇器 (Pseudo-classes)**：靈活運用 :hover、:focus 處理使用者交互。

* **變換與過渡 (Transform & Transition)**：實作平滑的水位升降與物體位移。
